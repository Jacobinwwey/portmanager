## Start Here

- If you are a human operator or maintainer, start with [Human](/en/human/).
- If you are building automation or non-interactive flows, start with [Agent](/en/agent/).
- If you need stable interfaces first, jump directly to [Reference](/en/reference/).
- If you are aligning implementation decisions, open [Architecture](/en/architecture/).
- If you need deployment and safety boundaries, read [Operations](/en/operations/).
- If you need sequencing, dependencies, and acceptance framing, open [Roadmap](/en/roadmap/).
