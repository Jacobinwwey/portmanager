## 从这里开始

- 如果你是人类操作者或维护者，先看 [Human](/zh/human/)。
- 如果你要做自动化或非交互接入，先看 [Agent](/zh/agent/)。
- 如果你要先看稳定接口，直接进入 [Reference](/zh/reference/)。
- 如果你要校准实现决策，进入 [Architecture](/zh/architecture/)。
- 如果你关心部署与安全边界，阅读 [Operations](/zh/operations/)。
- 如果你需要看实现顺序、依赖和验收目标，打开 [Roadmap](/zh/roadmap/)。
